About my project
Planning, problems and understanding of web-design

About my project:
My project is to create a website using HTML and CSS for my Computer science class, made up of at least 5 webpages to demonstrate my understanding of coding while creating something of personal interest to me.
We were provided with several categories from which I selected "Product Review".
I decided to create a website with book reviews, because I always loved to read and I personally think that it's helpful to get an opinion on a book while deciding whether it's worth reading.

Problems and solutions:
A good way to solve issues with HTML code is to first look for red error signs on the right of the code in WebStorm, those can be very helpful. If the code doesn't work, but has no error sign, it's very helpful to find a tutorial online for the piece of code that's problematic, for example on w3schools. W3schools is also a great tool to find the code for something you want to do with HTML, but don't know how to.
In my code I had a lot of Typos. Using WebStorm, those are relatively easy to find, because the program detects Typos for you. One of my most frequent problems was, that I wanted my website to do something, but I didn't know how to code it that way. I solved this issue, by searching a tutorial for the code I was looking for on w3schools.
I also had some problems setting up a CSS grid for my website. I used a YouTube tutorial on CSS grid to figure out what I was doing wrong. I also used the code from an example website created by my computer science teacher.

My Main page layout:

Layout:
For the planning of my website I looked at other book review websites on the internet to find an interesting layout and I combined the best elements.
I tried to make my layout for my main page and my content pages very organized and as similar as possible.

Time management:
I didn't really manage my time for my project,
I just tried to use class time efficiently and then worked on it on the weekend or in the afternoon whenever I liked to.

My Content page layout:

Color schemes:
For the color schemes I used on the different content pages I searched for a background pattern that fits the theme of the content page and created a color scheme using the colors in the background pattern.
On the main page and the about my project page I searched for a simple but fitting background that is not so colorful and used a very colorful color scheme to create some contrast.

Understanding of web-design:
I really like about my website, that it has a consistent layout, but different color schemes depending on the topic, that makes it more colorful. I also like that it has books from different genres, so there is something to read for everyone. I like that I included color gradients and hover animations to make the design more interesting.
I think people would want to visit my website, because it gives an organized overview about good books and helps them understand why they might want to read them.

I am not completely sure what level I would give myself for the website. I think I should have used a more consistent color scheme, because sometimes very colorful color schemes can distract from the actual content.
Maybe I should have used a simpler and more readable font for my text, because it might be hard to read for some people.
Considering the amount of things that I maybe should have done differently, I would probably grade myself with a level 3.
